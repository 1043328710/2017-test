position + margin

兼容性好，但是宽高需要已知

position: relative;
position: absolute; left: 50%; top: 50%; margin-left: -300px; margin-top: -50px;

position + transform

宽高不需要已知，但是兼容性不好

position: relative;
position: absolute; left: 50%; top: 50%; transform: translate(-50%, -50%);

position + margin

宽高不需要已知, 且兼容性好

position: relative;
position: absolute; left: 0; right: 0; top: 0; bottom: 0; margin: auto;

flex + margin

宽高不需要已知，未来趋势

margin: auto;
